# vim
vim config for c coder

## Init
* cp -r .vim* ~
* install ctags: `yum install ctags` or `apt-get install ctags` ...

## Keys
* **F5** `Update TagList`
* **F3** `Search Keywork`
* **F3 F3** `Close Search Window`
* **F2** `Highlight Keyword`
* **F2 F2** `Clear Highlight`
* **fu** `Search Function`
* **fp** `Search File`
* **tl** `Open or Close TagList`
* **bn** `Next Buffer Tab`
* **bp** `Prev Buffer Tab`
* **Q** `Close Current Buffer Tab`
